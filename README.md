# Season by Season NFL Win Loss Records (1920-Present)

Also includes AFPA and AFL records, annual records in separate files, and a combined csv file with all records separated by year, including modern team names for any franchises that are still functional. 

datagetter.py -> extracts tables from the .html files to save as .csv files in the ./csv folder. 
datacompiler.py -> combines all extracted data into one .csv file, and adds modern team names for currently operating franchises. 

**All data sourced from pro-football-reference.com.**
**No automated web scraping was used in the collection of this data, all web data collection was done manually in accordance with SR rate limits.**
**Anyone using this data should attribute all credits to them.**
